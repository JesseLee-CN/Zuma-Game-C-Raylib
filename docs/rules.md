# 祖玛游戏计分规则

## 设计原则

一次性消除的球越多，单球平均分值越高。分多次消除相同数量的球，总分应明显低于一次性消除。

## 计分公式

### 单轮消除基础分

每轮消除（一次连续的 ≥3 个同色球）的基础分为：

```
基础分 = N² × 10
```

其中 N 为该轮消除的球数（N ≥ 3）。

| 消除球数 | 基础分 | 单球均分 |
|:---:|:---:|:---:|
| 3 | 90 | 30.0 |
| 4 | 160 | 40.0 |
| 5 | 250 | 50.0 |
| 6 | 360 | 60.0 |
| 7 | 490 | 70.0 |
| 8 | 640 | 80.0 |

### 连锁加成

一次发射可能触发多轮消除（前一轮消除后，原本不相邻的同色球变为相邻，再次满足 ≥3 连条件）。每一轮连锁在基础分之上叠加连锁系数：

```
连锁系数(k) = 1 + (k − 1) × 0.5
```

其中 k 为连锁轮次（k = 1, 2, 3, ...）。

| 连锁轮次 | 系数 |
|:---:|:---:|
| 1 | ×1.0 |
| 2 | ×1.5 |
| 3 | ×2.0 |
| 4 | ×2.5 |
| 5 | ×3.0 |

### 单轮实际得分

```
单轮得分 = N² × 10 × (1 + (k − 1) × 0.5)
```

## 计分示例

### 示例 1：单次消除 6 球 vs 两次各消 3 球

- **一次性消除 6 球**（k=1）：
  ```
  得分 = 6² × 10 × 1.0 = 360
  ```

- **两次连锁各消 3 球**（k=1, k=2）：
  ```
  第 1 轮：3² × 10 × 1.0 = 90
  第 2 轮：3² × 10 × 1.5 = 135
  总分 = 90 + 135 = 225
  ```

360 > 225，一次性消除得分更高，满足设计原则。

### 示例 2：单次消除 8 球 vs 3+5 连锁

- **一次性消除 8 球**（k=1）：
  ```
  得分 = 8² × 10 × 1.0 = 640
  ```

- **先消 3 再连锁消 5**（k=1, k=2）：
  ```
  第 1 轮：3² × 10 × 1.0 = 90
  第 2 轮：5² × 10 × 1.5 = 375
  总分 = 90 + 375 = 465
  ```

640 > 465，依然一次性消除分更高。

### 示例 3：三次连锁

```
第 1 轮消除 3 球：3² × 10 × 1.0 = 90
第 2 轮消除 4 球：4² × 10 × 1.5 = 240
第 3 轮消除 3 球：3² × 10 × 2.0 = 180
总分 = 90 + 240 + 180 = 510
```

## 实现要点

1. `EliminateRuns` 每次消除一个 run 时，记录该轮的球数 N 和轮次 k，计算本轮得分并累加。
2. 一次发射触发的所有连锁视为同一次计分事件，发射球飞出边界或重新就位后结算。
3. 游戏界面需显示累计总分，位置建议在窗口左上角或右上角。
4. 每次消除发生时，可在消除位置短暂显示该轮得分（弹出式文字动画，约 500ms 后消失）。

## 后续扩展（可选）

- **连击计数**：记录连续成功消除的发射次数（不落空），连击数越高额外加成越大。
- **关卡目标分**：达到指定分数通关。
- **时间奖励**：在限定时间内完成消除有额外倍率。
