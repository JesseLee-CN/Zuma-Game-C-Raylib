# 祖玛游戏规则说明

## 基本玩法

- 球链沿阿基米德螺旋线排列，由外向内收缩
- 玩家瞄准并发射彩球，击中球链后球插入命中位置
- 连续 ≥3 个同色球被消除，并可能触发连锁消除
- 消除所有球即通关；射尽所有球后仍有剩余则失败

## 球颜色

共 6 种颜色：蓝、绿、红、黄、品红、棕。初始球链颜色按此顺序循环生成，发射球颜色随机。

## 难度设定

| 模式       | 初始球数 | 射击次数 |
| ---------- | -------- | -------- |
| INFINITE   | 10       | 无限     |
| EASY       | 10       | 100      |
| COMMON     | 20       | 80       |
| HARD       | 30       | 50       |
| CUSTOMIZED | 自定义   | 自定义   |

CUSTOMIZED 模式下，球数范围 5–50，射击次数范围 10–200，可切换为无限。

## 消除规则

一次发射后，从命中位置向两侧扫描，将 ≥3 个连续同色球作为一组（run）消除。消除后原本不相邻的同色球可能变为相邻，再次满足条件则触发连锁，循环至无可消除的组为止。

## 计分规则

### 单轮消除基础分

```
基础分 = N² × 10  （N ≥ 3）
```

| 消除球数 | 基础分 | 单球均分 |
|:---:|:---:|:---:|
| 3 | 90 | 30.0 |
| 4 | 160 | 40.0 |
| 5 | 250 | 50.0 |
| 6 | 360 | 60.0 |
| 7 | 490 | 70.0 |
| 8 | 640 | 80.0 |

### 连锁加成

一次发射触发的第 k 轮消除乘以连锁系数：

```
连锁系数(k) = 1 + (k − 1) × 0.5
```

| 连锁轮次 | 系数 |
|:---:|:---:|
| 1 | ×1.0 |
| 2 | ×1.5 |
| 3 | ×2.0 |
| 4 | ×2.5 |
| 5 | ×3.0 |

### 单轮实际得分

```
单轮得分 = N² × 10 × (1 + (k − 1) × 0.5)
```

一次性消球越多、连锁越深，总分越高。一次性消除 6 球（360 分）高于两次各消 3 球（90 + 135 = 225 分）。

## 剩余球惩罚

游戏结束时（通关失败或主动退出），每个剩余球扣除 **−20** 分。总分不会降至负数（最低为 0）。

## 游戏结束条件

| 条件 | 结果 |
| ---- | ---- |
| 球链全部消除 | 通关（CLEAR!） |
| 射击次数耗尽且仍有剩余球 | 失败（GAME OVER） |
| 发射球飞出窗口边界且射击次数耗尽 | 失败（GAME OVER） |
| PLAYING 期间点击 EXIT | 失败（GAME OVER），剩余球正常扣分 |

## 得分弹出

每次消除发生时，在消除位置短暂显示本轮得分（金色向上浮动）。结算阶段剩余球扣分时，在中央显示红色 `-20` 弹出文字。
